# IPL Data Analysis Dashboard

## 📊 Overview
This project analyzes IPL cricket data to extract insights on player performance and team statistics.

## 🔧 Tools Used
- Python
- Pandas
- Matplotlib
- Streamlit

## 📈 Key Insights
- Top run-scoring batsmen
- Leading wicket-taking bowlers
- Team win percentages
- Toss impact on match results

## 🚀 How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run dashboard:
   streamlit run app/app.py
